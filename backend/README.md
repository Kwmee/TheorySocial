# Backend

## Requisitos
- Java 17
- Maven 3.9+
- MySQL 8+

## Variables de entorno
- Copia `backend/.env.example` a `backend/.env` y ajusta credenciales.

## Arranque
```bash
mvn spring-boot:run
```

## Endpoints base
- `GET /api/users`
- `POST /api/users`
- `GET /api/theories`
- `POST /api/theories`
